1) pip install requirements.txt
2) Setup Sql Database using Commands in db_commands.txt
3) Intialize rasa in a directory using command "rasa init" and copy all the files to that directory 
5) run following code in terminal 
	a) rasa run --endpoints endpoints.yml --port 5005 --credentials credentials.yml --cors "*"
	b) rasa run actions 
	c) python web_app.py
